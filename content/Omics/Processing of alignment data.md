After completing the alignment the [[SAM;BAM files]] can be used to gain some insight into the quality of the alignment. The **Variant Call format** can be used to generate an overview over the differences between sample and reference sequence.![[Genomics_intro_KIMN20_-1822122806.pdf#page=38&rect=16,111,947,347|Genomics_intro_KIMN20_-1822122806, p.38]]
Another way to compare the sample and reference is the **Multiple sequence alignment** (MSA). ![[Genomics_intro_KIMN20_-1822122806.pdf#page=39&rect=76,185,747,325|Genomics_intro_KIMN20_-1822122806, p.39]]In this case a protein sequence is used, but this can be done with nucleotide sequences as well. The sequence is identified in the first *column* followed by the sequences, [[GAPs]] are introduced where there is variation between the sequences (e.g. due to INDELS in the DNA).

The **Tree** file format can be used to identify how closely related multiple sequences are to each other.
