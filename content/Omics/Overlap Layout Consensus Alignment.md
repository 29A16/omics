OLC is more used for the analysis of bigger genomes, using long read [[Denovo Assembly|sequences]]. The generated sequence reads are comapred against each other, requiering more complex processing than alignment via k-mers. The Consensus sequence or *“Contig”* is generated from the aligned reads, in cases of disagreement the majority rules.![[denovoAssembly_KIMN020.pdf#page=21&rect=213,25,715,380&color=yellow|denovoAssembly_KIMN020, p.21]]
