[[Metabolomics - Data processing]]
The data generated with a NMR is easier to analyse and follows these steps:
1. Baseline correction (multiple methods e.g. derivative)
2. Alignment — either global alignment with TSP standard or fine/local alignment
3. Feature detection — segmentation or peak based methods
4. Normalization — normally to total intensity
5. Peak identification — co-variation, 2D NMR, database.
