Clustering describes a procedure, where the variables of the data are projected onto a space of smaller dimension. This can be used to structure the data into groups aka clusters and the identification of outliers for [[Quality control of proteomics MS data|quality control]].
**Multidimensional scaling (MDS)** is one such technique, where a 2D plot takes into account he sample similarities to create clusters. 
Another technique is **Principal Component Analysis (PCA)** which creates components, that represent the biggest variation in the data and plots them against each other to create clusters.
### Supervised and Unsupervised Clustering
Clustering can be conducted with labelled reference data to create pre-defined groups/categories (supervised) or just with the raw data (unsupervised). Both approaches see use depending on the experimental data and the amount of prior knowledge — unsupervised is more exploratory.
